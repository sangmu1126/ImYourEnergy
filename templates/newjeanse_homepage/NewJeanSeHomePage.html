나만의 데이터 찾으러 가기

주요특징

ElectricKo! 프로젝트는 인공지능을 통해 각종 종류의 전력 수요를
예측하여 사용자가 원하는 맞춤 데이터를 지도 형태로 제공합니다.

커뮤니티 기능

예측 데이터 관련 회원 간 소통을 위한
다양한 커뮤니티 기능을 제공합니다.

사용자 시뮬레이션 지원

사용자가 맞춤 설정한 값을
기반으로 전력 수요를 예측하는
시뮬레이션 기능을 지원합니다.

직관적이고 아름다운 차트

한 눈에 이해하기 쉽고 아름다운 차트를 제공합니다.

다양한 예측 데이터 제공

사용자의 수요에 걸맞은 다양한 종류의
전력 수요 예측 데이터를 제공합니다.

회원 전용 데이터베이스 제공

회원 가입 시, 더욱 다양한 예측 데이터와
이용 컨텐츠들을 저장할 수 있는
나만의 전용 데이터베이스를 제공합니다.

데이터 전국 시군구 지원

제공하는 모든 데이터는 대한민국
전국 모든 시군구 단위로 지원합니다.
